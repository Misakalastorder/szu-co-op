# 手指结构要求
将手指结构按照urdf文件建模。
人手为右手，人手的掌心指向x轴正方向，大拇指指向y轴正方向，手指上方为z轴正方向。
根节点为手指腕部的节点，然后连接四根手指，分别为食指，中指，无名指，小指。
其中每根手指都有四个可动关节和一个指尖固定关节。第一个关节的旋转轴为x轴正方向即
（1,0,0），第二和第三个和第四个关节的旋转轴为y轴正方向即（0,1,0），第五个关节定义为固定关节无法旋转。
然后根节点还要连接大拇指，大拇指只有三个可动关节和一个固定关节，第一个关节的旋转轴为（1,0,0）第二和第三个关节的旋转轴为y轴正方向即（0,1,0）。
手指的相邻关节长度暂设置为0.04m。

# 数据手套说明文档
## 1. 数据内容数据中包含以下字段：
时间戳 

手指名称：thumb-拇指，index-食指，middle-中指，ring-无名指，pinky-小指 

AD：电压值 

angel：手指弯曲角度，范围 $0 \sim 90^\circ$ 

pitch：俯仰角（单位：$^\circ$） 

roll：翻滚角（单位：$^\circ$） 

yaw：偏航角（单位：$^\circ$） 

ax, ay, az：加速度（单位：$m/s^2$） 
## 2. 相关信号与参数说明该数据手套为右手使用。
手套中每个 IMU 的坐标系独立 。

（1）Angle（弯曲角度）物理含义：应变传感器的电信号反映的是手指中间关节和靠近指尖关节的弯曲角度总和 。简化模型：基于人手操作的协同性，通常认为这两个关节的弯曲角度一致 。例如，电信号显示 angel=30°，则认为两个关节各弯曲 $30^\circ$ 。精度与量程：标定范围为 $0 \sim 90^\circ$ 。角度不连续，分辨率为 $5^\circ$（如 $0^\circ, 5^\circ, 10^\circ \dots 90^\circ$） 。

（2）Pitch（俯仰角）正负定义：手指向上提起为正，向下压为负 。范围：理论范围为 $-180^\circ \sim 180^\circ$，实际操作范围一般为 $-100^\circ \sim 60^\circ$ 。

（3）Roll（翻滚角）范围：实际操作范围为 $-100^\circ \sim 100^\circ$ 。
